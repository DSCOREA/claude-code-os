## v1.0.35 — Xauthority + iwd + Alpine 3.20 build fixes

### Fixes
- `/home/cco/.Xauthority does not exist` — pre-create empty file (chmod 600). `.profile` adds fallback right before `exec startx`.
- `iwd failed to start` — added `/etc/iwd/main.conf` (`EnableNetworkConfiguration=false` + `NameResolvingService=none`, NetworkManager handles IP/DNS). `/var/lib/iwd` pre-created. `cco-infra.start` daemon spawns now `pgrep`-guarded so they don't race with the OpenRC services.
- Alpine 3.20 missing packages: dropped `wmctrl`, replaced `rfkill` with `util-linux-misc`.
- `setcap` multi-file syntax fails on Alpine 3.20 — split into one call per file with `|| true`.
- `/etc/local.d` missing in Alpine minirootfs — added `mkdir -p` before write + `rc-update add local default`.

### Install (one-liner, recommended)

Windows (PowerShell, admin):
```
iwr -useb https://raw.githubusercontent.com/Hostingglobal-Tech/claude-code-os/main/install-cco-on-ventoy.ps1 | iex
```

Linux / macOS:
```
curl -sL https://raw.githubusercontent.com/Hostingglobal-Tech/claude-code-os/main/install-cco-on-ventoy.sh | bash
```

Ventoy must be installed on the USB first (https://www.ventoy.net).
